# Requirements
- Python 3.11.x
```
pip install requests valclient
```
**BOTH** `instalock.py` and `data.json` are **REQUIRED**

# Regions
**Short code** is meant to be used, not Full Name.

| Short Code | Full Name |
| ------ | ------ |
| AP | Asia-Pacific |
| EU | Europe |
| KR | Korea |
| BR | Brazil |
| NA | North America |
| LATAM | Latin America |
